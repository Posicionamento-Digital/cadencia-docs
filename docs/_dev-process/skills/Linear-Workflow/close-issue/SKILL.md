# close-issue

Skill para fechar uma issue do Linear.

Consultar: `C:\Users\felip\.claude\skills\close-issue\SKILL.md`

---

**Uso:** `/close-issue PDL-11` ou `/close-issue` (lista In Progress)

**O que faz:**
1. Marca a issue como Done no Linear
2. Mostra template de commit com `Closes PDL-XX`
3. Verifica se tem arquivos não commitados antes de fechar
