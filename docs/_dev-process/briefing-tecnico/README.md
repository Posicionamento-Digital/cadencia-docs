# Processos — Briefing Técnico

Template e processo de briefing técnico para novos clientes da PD. Executado entre o fechamento do contrato e o kickoff de implementação.

## O que fica aqui

- Template de briefing técnico (formulário + reunião)
- Checklist de informações necessárias por tipo de produto
- Como conduzir a reunião de briefing (pauta, perguntas-chave)
- Como documentar decisões do briefing para o time técnico
- Handoff do comercial para a implementação

## Informações coletadas no briefing

| Produto | O que coletar |
|---|---|
| Agente de atendimento | Tom de voz, FAQ, horários, responsável |
| CRM / GHL | Funis atuais, etapas, responsáveis |
| SDR com IA | ICP, mercado, script atual, objeções |
| Cadência (carrossel) | Nicho, público, temas, paleta |
