# Runbooks — GCI-GO

Procedimentos operacionais para a implementação GCI-GO. Usados pela equipe PD para resolver situações sem precisar escalar para Felipe.

## O que fica aqui

- Como reconectar instância WhatsApp desconectada (geração de QR Code)
- Como abrir chamado de suporte no HUB|PD corretamente
- Como verificar se a Lara está respondendo e o que fazer se parou
- Como trocar profissional e revogar acesso de usuário desligado
- Como verificar disponibilidade no Ecuro quando agendamento falhar
- Como atualizar scripts da Lara após mudança de protocolo da clínica

## Convenção de nomes

`RB-[numero]-[descricao-slug].md` — ex: `RB-01-reconectar-whatsapp.md`
